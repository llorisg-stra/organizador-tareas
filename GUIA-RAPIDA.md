# ⚡ Guía Rápida - Cheatsheet

Referencia de una página con lo esencial. **24 comandos disponibles.**

---

## 🚀 Comandos más usados

| Comando | Qué hace |
|---------|----------|
| `/inicio` | Empezar a trabajar |
| `/fin` | Terminar y sincronizar |
| `/ayuda` | Ver todos los comandos |
| `/contexto` | Gestionar notas/estado |
| `/tareas` | Ver pendientes |
| `/documento` | Añadir PDF, imagen, DOCX |

---

## 📁 Archivos importantes

| Archivo | Para qué |
|---------|----------|
| `CONTEXT.md` | Estado del proyecto (⭐ el más importante) |
| `README.md` | Descripción del proyecto |
| `.agent/workflows/` | Comandos personalizados |
| `docs/conocimiento/REGISTRO.md` | Índice de documentos externos |

---

## 🔄 Flujo diario

```
/inicio  →  trabajar  →  /fin
```

---

## 📂 Nuevo proyecto

```
1. Copiar _PLANTILLA-PROYECTO
2. Renombrar
3. Editar README.md y CONTEXT.md
4. Borrar carpetas innecesarias
5. "Inicializa Git"
6. Crear repo en GitHub
7. "Conecta con [URL]"
8. /inicio
```
