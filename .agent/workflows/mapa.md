# Mapa del Proyecto

Muestra un resumen visual de todas las secciones y su estado.

## Pasos

1. **Escanear carpetas del proyecto**

2. **Mostrar mapa estructurado**:
   ```
   📂 PROYECTO
   
   💻 Desarrollo
   ├── backend/     [X archivos] 
   ├── frontend/    [X archivos]
   ├── n8n/         [X workflows]
   └── deploy/      [configurado/pendiente]
   
   📚 Documentación
   ├── docs/apis/   [X APIs documentadas]
   └── docs/diseño/ [X mockups]
   
   💼 Negocio
   ├── marketing/   [activo/inactivo]
   └── legal/       [completo/pendiente]
   
   ... etc
   ```

3. **Identificar secciones vacías** y sugerir si son necesarias

4. **Ofrecer**: "¿Quieres trabajar en alguna sección específica?"

## Notas
- Útil para tener visión general del proyecto
- Ayuda a identificar áreas pendientes
