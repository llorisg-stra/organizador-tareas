# Ver Todo el Contexto

Muestra el CONTEXT.md completo con formato legible.

## Pasos

1. **Leer archivo CONTEXT.md** completo

2. **Mostrar contenido formateado**:
   - Objetivo del proyecto
   - Estado actual con checkboxes
   - Todas las decisiones técnicas
   - Stack completo
   - Recursos y enlaces
   - Historial de notas
   - Próximos pasos

3. **Mostrar estadísticas**:
   ```
   📊 Resumen:
   - Última actualización: [fecha]
   - Decisiones técnicas: [X]
   - Notas registradas: [X]
   - Tareas pendientes: [X]
   - Tareas completadas: [X]
   ```

4. **Ofrecer acciones**:
   - ¿Quieres actualizar algo? → /contexto
   - ¿Quieres ver el mapa? → /mapa
   - ¿Quieres empezar a trabajar? → Indicar área

## Notas
- Este workflow solo MUESTRA, no modifica
- Para modificar usar /contexto
- Si el CONTEXT.md no existe, ofrecer crearlo
