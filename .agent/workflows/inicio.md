# Inicio de Sesión

Lee el estado del proyecto y muestra un resumen para empezar a trabajar.

## Pasos

1. **Leer archivo CONTEXT.md** del proyecto actual

2. **Mostrar resumen estructurado**:
   - Nombre y objetivo del proyecto
   - Estado actual (fase)
   - Últimas notas/decisiones
   - Próximos pasos pendientes

3. **Preguntar al usuario**: "¿En qué quieres trabajar hoy?"
   - Ofrecer opciones basadas en los próximos pasos
   - O permitir que indique otra tarea

## Notas
- Este workflow se usa al inicio de cada sesión de trabajo
- Ayuda a retomar el contexto rápidamente
- Si CONTEXT.md no existe, ofrecer crearlo con /contexto
