## 🔍 Glosario de Términos

| Término | Definición |
|---------|------------|
| **Antigravity** | La herramienta de IA para desarrollo |
| **Workflow** | Comando personalizado (ej: `/inicio`) |
| **CONTEXT.md** | Archivo que guarda el estado del proyecto |
| **Perfil** | Espacio de trabajo separado (personal/empresa) |
| **Plantilla** | Carpeta `_PLANTILLA-PROYECTO` base para nuevos proyectos |
| **Sync** | Sincronizar cambios con GitHub |
| **Workspace** | Carpeta raíz de un proyecto |
| **Base de Conocimiento** | Carpeta `docs/conocimiento/` con documentos externos |
| **REGISTRO.md** | "Base de datos" de documentos externos añadidos |
| **Resumen** | Archivo Markdown con extracto de un documento externo |

## 🔢 Comandos Rápidos (24 total)

### Flujo básico
```
/inicio     → Empezar sesión
/fin        → Terminar y sincronizar
/ayuda      → Ver todos los comandos
```

### Los más usados
```
/contexto   → Gestionar notas/estado
/tareas     → Ver pendientes
/git-sync   → Sincronizar con GitHub
/mapa       → Ver estructura
```

### Documentación
```
/documento  → Añadir PDF, DOCX, imagen al proyecto
/decision   → Registrar decisión técnica
/api        → Documentar API
/meeting    → Notas de reunión
```
