# 🧪 Testing

Documentación y recursos de testing.

## Tipos de Tests

| Tipo | Herramienta | Cobertura |
|------|-------------|-----------|
| Unit | Jest/Vitest | % |
| Integration | - | % |
| E2E | Playwright/Cypress | % |
| Manual | - | - |

## Comandos

```bash
# Ejecutar todos los tests
npm test

# Tests con cobertura
npm run test:coverage

# Tests E2E
npm run test:e2e
```
